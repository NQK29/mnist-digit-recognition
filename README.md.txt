# Handwritten Digit Recognition (MNIST)

A deep learning project to classify handwritten digits (0–9) using the MNIST dataset.

## 🚀 Features
- Preprocess the MNIST dataset (60,000 training and 10,000 test images).
- Build and train a Convolutional Neural Network (CNN) with TensorFlow/Keras.
- Evaluate accuracy on test set.
- Predict custom handwritten digit images.

## 🛠 Tech Stack
- Python 3.x
- TensorFlow / Keras
- NumPy, Matplotlib
